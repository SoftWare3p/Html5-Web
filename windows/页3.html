如何使用 Windows 10

寻找新的方法，更快完成工作？查看这些 Windows 10 提示和技巧，它们可以帮助您充分利用电脑和时间！

功能提示

赋予您的电脑时尚新外观

通过切换到浅色或深色模式来更改电脑。选择开始>设置>个性化>颜色。在选择您的颜色下，选择浅色或深色模式。

了解详细消息

从时间线中返回过往活动

要打开时间线，请选择任务栏上的任务视图，或同时按下Windows 徽标键和 Tab 键。向下滚动，直到找到您想要返回的记录，然后单击即可从以前停止的地方恢复。

截取屏幕上的内容

按下 Windows 徽标键+Shift 键 + S 键打开截图栏，然后将光标拖到想要捕获的区域。您截取的区域将保存到剪贴板。

快速查找照片

在照片中搜索人员、地点、事物和文字。您还可以找到收藏夹、特定文件或文件夹。“照片”会为您添加标签，因此，您不需要无休止地滚动，即可找到所需内容。

并排分屏的应用

选择任何打开的窗口，然后将其拖动并弹向屏幕侧面。您打开的所有其他窗口将显示在另一侧。选择一个窗口，使其填满开放空间。

通过 Microsoft 帐户充分利用 Windows 10

使用 Microsoft 帐户登录，可在 Windows 10 电脑上获得丰富的功能。

Microsoft 的新增功能

开始使用 Microsoft 产品等。

安全性

更改密码，更新安全信息，并将重要的帐户详细信息保持最新。

隐私

查看您的搜索历史记录、浏览和定位活动等。

家庭

即使与家人分开，也可以确保家人上网安全并保持联系。
